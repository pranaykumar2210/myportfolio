INTERNET OF THINGS (IOT) FOR BANK LOCKER SECURITY SYSTEM

This article presents the design and implementation of an IoT based bank locker security system utilizing the ESP32-CAM microcontroller and Telegram messaging platform. By installing a camera, wireless connectivity and an instant messaging module, the system aims at providing security and convenient door access control. ESP32-CAM is recording images of persons entering the door, sending a notification via telegram to authorized users. This system allows users to control lockers via remote access from anywhere with the Internet connection, monitor entrance and permit or reject access at any time.
